Projects

SQL: Seattle Construction Trends and Development Analysis

This SQL project explores Seattle's transformation over the past decade, analyzing where the city is growing, which dwelling types are dominating new construction, and how neighborhoods are evolving through demolition and redevelopment. By examining permits across zoning classifications and neighborhoods, the analysis uncovers trends in residential and commercial development from 2012 to present. This project showcases the ETL process in Python, SQL queries, and Tableau visualizations.

Python: Credit Card Fraud Detection with Machine Learning Models

This machine learning project classifies credit card transactions as fraud or not fraud using European cardholder data. By training and comparing four classification models—Logistic Regression, Decision Tree, Random Forest, and XGBoost—the analysis identifies which algorithms most accurately detect fraud. This project showcases Python, machine learning classification, model evaluation, and data visualization (matplotlib, seaborn). The data has been anonymized to protect the cardholders' identities.
